Each API that has the Protected set to 'Yes' need the JWT in order to accomplish the request. So, fisrt thing to do is to call /api/auth/login in order to obatin your token. Token then must be passed in each request into the header 'Authorization' AS IS.

POST No /api/auth/login Login the provided username, by returning a JSON Web Token. The token will be used in each other request, and it must be used 'as is'.

/api/notes/create
PUT Yes /api/notes/create Create a new note for the logged in user with the provided tags. Tags must be already created. This API requires the tags (mainTags and otherTags) to be an array of ID. Obviously, the tags must be for the logged in user.
  • Authorization: token
  • Content-type: application/json
 A JSON object:
{"note":{"title":"note-title","text":"note-text","mainTags":[array-of-ids], "otherTags":[array-of-ids]}}

{"note": {..."links":["uri","uri"], "isDone":true}} 		If ok:
